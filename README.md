# Zhuamao

Express + Hogan + React + Redux + Gulp + Webpack

=====

###dev:
**install packages**
sudo npm i

**run server**
supervisor -- app/app.js -p 8001

**build third-party js/css libs**
gulp third-party

**build base js/css files**
gulp webpack:build -e base

**build your works**
gulp webpack:dev -e adminHomepage

**preview**
http://0.0.0.0:8001/admin/
